Политика конфиденциальности

Дата вступления в силу: 01.01.2025

Приложение: Smart Health Home

1. Какие данные мы собираем

Наше приложение получает доступ к следующим данным о здоровье через Health Service Kit:

  • ✅ Количество шагов
  • ✅ Пульс (частота сердечных сокращений)
  • ✅ Данные о сне (продолжительность, качество)
  • ✅ Сожженные калории
  • ✅ Уровень стресса
  • ✅ Артериальное давление (опционально)
  • ✅ Уровень кислорода в крови (SpO2)
  • ✅ Температура тела
  • ✅ Пройденная дистанция
  • ✅ Активные часы

2. Как мы используем данные

Собранные данные используются для:

  • 📊 Анализа состояния здоровья
  • 🏠 Автоматизации умного дома (освещение, температура, уведомления)
  • 📱 Персонализированных рекомендаций
  • 📈 Отслеживания прогресса здоровья

3. Хранение данных

  • Данные хранятся локально на вашем устройстве
  • Мы не передаем данные третьим лицам без вашего согласия
  • Данные синхронизируются с Health Service Kit (Google Fit / Huawei Health)
  • Для работы умного дома данные передаются в Home Assistant по защищенному соединению

4. Безопасность

  • 🔒 Все передачи данных зашифрованы (HTTPS/TLS)
  • 🔑 Доступ к умному дому защищен токенами авторизации
  • 🛡️ Мы не храним ваши учетные данные

5. Права пользователей

Вы имеете право:

  • ✅ Отозвать доступ к данным в любой момент
  • ✅ Удалить все собранные данные
  • ✅ Экспортировать свои данные
  • ✅ Отключить интеграцию с умным домом

6. Третьи стороны

Приложение взаимодействует со следующими сервисами:

  • Google Fit API / Huawei Health Kit - получение данных о здоровье
  • Home Assistant - управление умным домом

Эти сервисы имеют свои политики конфиденциальности.
